# 📝 Notes App

A simple and modern Flutter Notes application that allows users to create, edit, delete, and organize notes with a clean and responsive user interface.

## ✨ Features

- 📝 Create new notes
- ✏️ Edit existing notes
- 🗑️ Delete notes
- 🎨 Choose note colors
- 💾 Local data storage using Hive
- ⚡ State management using Flutter Bloc (Cubit)
- 📱 Responsive and clean UI

## 🛠️ Tech Stack

- Flutter
- Dart
- Hive
- Flutter Bloc (Cubit)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Tahasami12/notes.git
```

2. Install dependencies:

```bash
flutter pub get
```

3. Run the project:

```bash
flutter run
```
